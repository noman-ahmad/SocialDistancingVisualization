# SocialDistancingVisualization
Tableau Visualization of Social Distancing Effects on the Rate of  COVID-19 in New York City.
- This Projects Analyzes The Effetcs of Social Distancing in New York City during the COVID-19 Pandemic. 
- We used Tableau as a Visualization Tool To Analyze and Display these Effects. 
- This Tableau Visualization combines maps, tables, and graphs to show social distancing from various different analytical perspectives. 
- In order to represent social distancing, we used data from New York City Parks. 

Usage Instructions: 
- after cloning/forking, download the wokbook "Social Distancing in Parks.twbx" from the workbooks directory (the only file) 
- Open the workbook in Tableau
- Interact with the different maps, tables, graphs in our workbook. 

Links to Data Sets Used (found in data directory): 
- Park Encounters :  https://data.cityofnewyork.us/dataset/Social-Distancing-Parks-Ambassador-Data/98ef-fxnm
- Park Patrol     :  https://data.cityofnewyork.us/dataset/Social-Distancing-Parks-Crowds-Data/gyrw-gvqc   
- Covid-19 By Zip :  https://www1.nyc.gov/site/doh/covid/covid-19-data.page  (original Link no longer accesible but this is the source)
- NYC Park Zones  :  https://nycopendata.socrata.com/City-Government/Parks-Zones/4j29-i5ry

Project File Structure: 
- data      : holds all the data we used, as well as edited data (which removes some columns not important in our analysis/visualiztion, no changes were made to value of data)
- scripts   : holds the python scripts we used to edit data, as well as analyze our data
- workbooks : holds single .twbx file containing our visualization, to be used with Tableau Application.
